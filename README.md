### Dependencies

* node.js v16.16.0


### Installation

1. In the directory of the server (./server):
  ```sh
  npm install
  ```
2. In the directory of the client (./client):
  ```sh
  npm install
  ```
<p align="right">(<a href="#top">back to top</a>)</p>
  
  
  
### How to start

1. In the directory of the server (./server):
  ```sh
  npm start
  ```
2. In the directory of the client (./client):
  ```sh
  npm run serve
  ```
